## Projects

### 03 — Movie Recommendation System
Content-based recommender built on the MovieLens dataset using Cosine Similarity on one-hot encoded genre vectors.

- Dataset: 9,742 movies · 100,836 ratings
- Encodes 19 genres using one-hot encoding
- Returns Top 10 recommended movies with similarity scores and average ratings
- Visualizes results with dual bar charts

Tech: Python · Scikit-learn · Pandas · Matplotlib

### 02 — Iris Classification
Multi-class classifier on the Iris dataset using Decision Tree.

- Dataset: 150 samples · 4 features · 3 classes
- Algorithm: Decision Tree (max_depth=3)
- Test Accuracy: 96.67% on 30-sample holdout
- Includes confusion matrix and classification report

Tech: Python · Scikit-learn · Pandas · Matplotlib · Seaborn

### 01 — Rule-Based Chatbot
Terminal chatbot using keyword matching and conditional logic.
Handles greetings, time/date queries, basic arithmetic, and jokes.
No ML involved — demonstrates Python control flow and program structure.

Tech: Python
