# Iris Classification Model

A supervised machine learning model that classifies Iris flowers into 3 species.

## Features
- Loads Iris dataset from sklearn
- Splits data into 80/20 train-test sets
- Trains a Decision Tree Classifier
- Evaluates with accuracy, classification report, and confusion matrix heatmap

## How to Run
```
jupyter notebook iris_classification.ipynb
```

## Tech Stack
- Python
- scikit-learn
- pandas
- matplotlib, seaborn
